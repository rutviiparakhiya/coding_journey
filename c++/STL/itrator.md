#_itrator :
An iterator is an object that is used to point to and move through elements of a container.

1. Input Iterator : It is used to read elements from a container one by one in forward direction.

2. Output Iterator : It is used to write elements to a container one by one.

3. Forward Iterator : It is used to read or write elements while moving only forward in a container.

4. Bidirectional Iterator : It can move forward and backward through elements of a container.

5. Random Access Iterator : It can directly access any element in a container using an index.
